# Change Log

All notable changes to the "Scoped Refactor" extension are documented in this file.

## [Unreleased]

- Renamed display branding to `Scoped Refactor`.
- Added multi-action workflow:
  - `Add before to...`
  - `Remove all...`
  - `Edit all...`
- Added target list controls (`+` and `-`) with at least one field always available.
- Added path rule list controls (`+` and `-`) with two modes:
  - `Ignore path`
  - `Only modify this path`
- Added result preview via `Validate results`.
- Added in-view result panel (bottom text box) instead of corner-only notifications.
- Added comment-safe matching (`//`, `/* ... */`, `#`).
